# modelplane.ai website

Marketing website for [Modelplane](https://modelplane.ai), built with Next.js and deployed on Vercel.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deploys automatically on push to `main` via Vercel.
