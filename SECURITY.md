# Security Policy

This repository hosts the Modelplane website and blog.

To report a security issue with the site, please use GitHub's
[private vulnerability reporting][advisory] (the **Security** tab →
**Report a vulnerability**) rather than opening a public issue.

For vulnerabilities in the Modelplane software itself, follow the security
policy in the main repository:
[github.com/modelplaneai/modelplane](https://github.com/modelplaneai/modelplane/security/policy).

[advisory]: https://github.com/modelplaneai/website/security/advisories/new
